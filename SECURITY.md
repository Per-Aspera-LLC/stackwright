# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅                 |
| 0.x     | ⚠️ Security only  |

## Reporting a Vulnerability

For sensitive security issues, please email security@per-aspera.dev

For non-sensitive issues, please open a GitHub Security Advisory:
https://github.com/Per-Aspera-LLC/stackwright/security/advisories/new

## Response Timeline

- **Initial Response**: 24-48 hours
- **Severity Assessment**: 48-72 hours  
- **Fix Released**: Critical: 7 days, High: 30 days, Medium: 90 days

## Known Vulnerabilities

See our [Dependabot Security Advisories](https://github.com/Per-Aspera-LLC/stackwright/security/dependabot)
